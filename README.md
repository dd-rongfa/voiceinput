## 3. 缓冲区也吸收改口与练习表达

本人未提交 22,237 条里，覆盖 6,002 条（27.0%），超时丢弃 10,483 条（47.1%）。这说明缓冲区不只是过滤噪声，也在吸收改口、重说，以及一部分“说出来但不一定要提交”的表达过程。

完整报告里保留了一个谨慎判断：未提交不全是失败，其中可能包含练习表达、自言自语和口头梳理；这部分现象保留，但旧版那组更激进的衍生指标不再放在首页主叙述里。

## 4. 表达方式影响成功率

最稳的不是蹦词，也不是长篇口述，而是 2–5 秒、6–30 字的完整短句。
